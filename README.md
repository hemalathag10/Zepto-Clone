# Zepto Clone 

A full-stack grocery delivery application inspired by Zepto.

---

## 🚀 Tech Stack

### Backend
- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

### Frontend (planned)
- Angular

---

## 🗄️ Database
- MySQL used
- Database name: zepto_db

---

## ⚙️ How to run backend

cd zepto-backend
mvn spring-boot:run
